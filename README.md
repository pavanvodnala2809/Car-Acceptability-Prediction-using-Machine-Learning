# Car Acceptability Prediction using Machine Learning

## Project Overview

This project predicts car acceptability based on features such as buying price, maintenance cost, safety level, luggage capacity, and number of persons.

## Dataset

Car Evaluation Dataset
Total Records: 1728
Features: 6
Target: Outcome

## Steps Performed

Data Cleaning
Handling missing values
Encoding categorical data
Exploratory Data Analysis
Random Forest Model
Model Evaluation
Feature Importance Analysis

## Machine Learning Model

Random Forest Classifier

## Results

Model Accuracy: 94% (approx)

## Visualizations

Confusion Matrix
Feature Importance Plot

## Technologies Used

Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn

## How to Run

Install dependencies:

pip install -r requirements.txt

Run project:

python main.py

## Key Insights

Safety feature had the highest impact on car acceptability.
Number of persons was the second most important factor.
Buying price had moderate influence.
